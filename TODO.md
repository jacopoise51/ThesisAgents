## How to use

1. Add a task under your section as a checkbox: `- [ ] short title`.
2. Below it, paste the exact prompt you want Claude to run, as a
   blockquote (lines starting with `>`). Be specific — file paths,
   constraints, what NOT to do.
3. To run a task: open Claude Code in the repo root and paste the
   blockquote verbatim.
4. When done, tick the box (`- [x]`) and commit. Leave the prompt in
   place so we have a record of what was actually run.
5. If a task is no longer relevant, delete it rather than leaving it
   unchecked.

Owners are starting suggestions, not contracts — re-shuffle freely.
